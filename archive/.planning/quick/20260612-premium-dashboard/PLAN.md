# Stage 15 Plan - Premium Dashboard

## Goal

Redesign the dashboard into a premium, owner-friendly SaaS home screen that gives immediate context, actions, work requiring attention, and recent finance records.

## Scope

- Add a welcome section with user name, role, company, current date, and GST time.
- Add compact KPI cards for revenue, outstanding, cash received, expenses, VAT due, and net profit.
- Add quick actions for invoice, payment, expense, customer, and supplier workflows.
- Add recent activity feed using real audit events.
- Add pending approvals for submitted invoices and draft/approval-stage expenses.
- Add recent records for invoices, customers, expenses, and payments.
- Improve empty states with guided actions.
- Reuse existing dashboard data and routes.

## Out of Scope

- Full global search.
- Notification backend.
- Page-by-page table redesign.

## Verification

- TypeScript check
- Lint
- Production build
